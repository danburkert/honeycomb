Detailed Description

A serializable container that stores metadata for a MySQL table's indices and columns. For example, a MySQL table:

create table foo (x int, index (x));

will have a TableSchema containing one ColumnSchema and IndexSchema. The ColumnSchema will contain information such as the type int and column name x. The IndexSchema will contain information such as a uniqueness constraint and column name.
